# Blog Post Creation

Write a complete blog post based on the provided topic or outline.

## Process

1. **Research**: Use web fetch to gather current information on the topic
2. **Outline**: Create structure following the blog post template
3. **Draft**: Write the post in simple, concise English
4. **Validate**: Cross-reference facts with 2+ sources
5. **Review**: Send to @brutal-critic for quality scoring

## Requirements

- Under 800 words
- Simple English (under 20 words per sentence)
- Active voice throughout
- Include 3+ source links
- Working code examples for tech posts
- SEO-optimized title (under 60 characters)
- Clear takeaways and call-to-action
