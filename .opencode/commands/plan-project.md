# Project Planning & Orchestration

Create a comprehensive plan for the described project, then coordinate execution across specialized agents.

## Planning Phase

1. **Understand**: Clarify goals, constraints, and success criteria
2. **Analyze**: Review current codebase state and affected components
3. **Research**: Gather best practices and identify challenges
4. **Plan**: Create detailed phased plan with agent assignments

## Plan Template

For each phase, specify:
- **Agent**: Which specialized agent handles it (@codebase, @review, @docs)
- **Tasks**: What needs to be done
- **Dependencies**: What must complete first
- **Deliverables**: Expected outputs
- **Validation**: How to verify success

## Execution

After plan approval:
- Execute phases sequentially with validation gates
- Hand off to appropriate agents with full context
- Monitor completion and integrate outputs
- Provide final summary with deliverables
